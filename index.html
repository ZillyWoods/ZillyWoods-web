ZillyWoods Client
About
Custom client for Teeworlds version 0.7+, with basic DDRace modification features such as:
  • Entities overlay - cl_overlay_entities <0|100>
  • Custom zoom - zoom+, zoom, zoom-
  • Visible hook collision lines - bind <key> +showhookcoll (on key hold)
  • DDRace tiles prediction (hookthrough & stopper)
  • DDRace teams colored in scoreboard
  • *Experimental* DDRace map editor support (switch, front, tele & speedup layers)
  • Scoreboard and statboard support 64 players
  • Highlight server administrators (with red color) in scoreboard - cl_show_admins <0|1>
  • Connect dummy button
  • Dummy switch bind - bind <key> "toggle cl_dummy 1 0"
  • Info about spectators leaving the server - cl_show_silent_messages <0|1>
  • Show prediction to server - cl_showpred <0|1>

For more info and older releases visit GitHub
